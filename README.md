# YouTube Playlist Tracker Application

## Overview
The YouTube Playlist Tracker Application allows users to track and manage their favorite YouTube playlists efficiently. With an intuitive interface and powerful features, this application is designed to enhance user experience by providing easy access to playlist details and updates.

## Features
- **Track Playlists**: Users can add and track multiple playlists from their YouTube account.
- **Real-time Updates**: The application fetches and displays the latest videos in the tracked playlists automatically.
- **Search Functionality**: Easily search for playlists or specific videos within playlists.
- **User Authentication**: Secure login and user management to personalize the experience.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## Technologies Used
- **Frontend**: React.js, HTML, CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **APIs**: YouTube Data API for accessing playlist and video information.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/aman-shahi-dev/Bookmarker.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Bookmarker
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Set up environment variables for YouTube API credentials.
5. Start the application:
   ```bash
   npm start
   ```

## Usage
Once the application is running, users can log in and start tracking their playlists with just a few clicks. The interface guides users through the process of adding and managing playlists.
